# 📰 News App - ReactJS

A modern web application for real-time news aggregation and visualization. This project fetches news data from external APIs and presents it in a clean, responsive user interface built with React.

---

## 🚀 Features

- 📡 Real-time news fetching from external APIs
- 🗂️ Categorized news (e.g., Technology, Sports, Business, etc.)
- 🔍 Search functionality for articles
- 📱 Responsive design (mobile & desktop)
- 🔗 Direct access to full articles via external links
- ⚡ Fast and dynamic UI using React

---

## 🛠️ Technologies Used

- **ReactJS** – Frontend framework
- **JavaScript (ES6+)** – Core programming language
- **CSS** – Styling and layout
- **REST APIs** – News data source
- **Axios / Fetch API** – HTTP requests

---

## 📦 Installation

1. Clone the repository:

```
git clone https://github.com/bardiaAsraary/News_app.git
cd News_app
```
## Install dependencies:
```
npm install
```
Start the development server:
```
npm start
```
The app will run on:
```
http://localhost:3000
```
## 🧩 Project Structure
```
src/
│── components/     # Reusable UI components
│── pages/          # Page-level components
│── services/       # API calls and data fetching
│── App.js          # Main app component
│── index.js        # Entry point
```
## ⚙️ How It Works
The application sends HTTP requests to a news API
Data is fetched asynchronously
React components dynamically render news articles
Users can browse by category or search for specific topics
